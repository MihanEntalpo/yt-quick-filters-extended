# Как установить расширение вручную

Эта версия предназначена для установки через режим разработчика в браузере Chrome или Chromium.

## Что нужно сделать

1. Распакуйте архив с расширением в любую удобную папку.
2. Откройте в браузере страницу:

   `chrome://extensions/`

3. Включите переключатель **Режим разработчика** (`Developer mode`) в правом верхнем углу.
4. Нажмите кнопку **Загрузить распакованное расширение** (`Load unpacked`).
5. Выберите папку, в которой лежат файлы расширения.

Важно: выбирать нужно именно папку, внутри которой находятся:

- `manifest.json`
- папка `dist`
- папка `public`
- папка `icons`

## Если расширение уже было установлено раньше

После обновления файлов:

1. Откройте `chrome://extensions/`
2. Найдите расширение `YouTrack Quick Filters Extended`
3. Нажмите кнопку **Обновить** / **Reload**
4. Обновите вкладку YouTrack

## Если браузер ругается

- Убедитесь, что вы выбрали правильную папку, а не родительский каталог.
- Убедитесь, что внутри выбранной папки есть файл `manifest.json`.
- Если расширение не обновилось, нажмите `Reload` на странице `chrome://extensions/`.

## Как удалить расширение

1. Откройте `chrome://extensions/`
2. Найдите `YouTrack Quick Filters Extended`
3. Нажмите **Удалить** (`Remove`)
